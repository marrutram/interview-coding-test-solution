# Configuration

## Environment Variables:

| NAME          | REQUIRED | DEFAULT                                         | DESCRIPTION                   |
| ------------- | :------: | :---------------------------------------------: | :--------------------------:  |
| NODE_END      | NO*      | development                                     |  |
| HOSTNAME      | NO*      | localhost                                       | Name server                   |
| PORT          | NO*      | 3000                                            | Port to start the app         |

# init
npm install

# start test

npm run test

# start test coverage report

npm run test-reporter

# start

docker-compose up
